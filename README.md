# password-generator-and-saver


🔑 Local Password Manager
This project is a lightweight, secure-by-simplicity utility that automates the creation and storage of strong credentials. Instead of relying on memory or repetitive passwords, this tool generates unique strings and catalogs them alongside their respective website addresses in a local text-based repository. It focuses on a "one-click" workflow to bridge the gap between generating a secure key and ensuring it is never lost.

🚀 Key Technical Highlights
Automated Generation: Uses randomized logic to build complex passwords, significantly increasing account security compared to manual entries.

Persistent Storage: Features an integrated filing system that appends new credentials and URLs directly to a .txt file for offline access.

Workflow Efficiency: Combines the generation and saving steps into a single action, reducing user friction and ensuring every new password is automatically backed up.
